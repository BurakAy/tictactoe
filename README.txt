Site: www.burakaydemir.com
Project: Tic Tac Toe Game

Created with HTML, JavaScript, and jQuery. 

Player can select to be either X or O at start of each game.
Computer selects random empty space for move.
Store moves into array and check function to see if winning combination is present. If so, update score and reset board, else continue playing.
